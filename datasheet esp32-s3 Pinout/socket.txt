use socker header pin 30 for all cubesat system ;

A (A1 → A15) :
Pin	Net/Signal what it use for?
A1	GND_STACK	ground
A2	VBAT_STACK	bat 7.4V send to Powerboard
A3	GND_STACK	ground (return current of VBAT)
A4	VBAT_STACK	bat 7.4V parallel add current/reduce voltage drop
A5	GND_STACK	ground
A6	3V3_STACK	3.3V from Powerboard send to all board
A7	GND_STACK	ground
A8	3V3_STACK	parallel add current
A9	GND_STACK	ground 
A10	I2C_SCL (GPIO42)	go to INA226/RTC/sensor
A12	I2C_SDA (GPIO41)	go to INA226/RTC/sensor
A11	GPS_TX (GPIO21)	ESP receive data (connect to GPS RX) // GPS TX -> ESP RX
A13	GPS_RX (GPIO47)	ESP send (connect to GPS TX) // GPS RX -> ESP TX
A14	PWR_EN / BUCK_EN (optional) command to open/close buck or open system (if you want)
A15	GND_STACK	ground

B (B1 → B15) :
Pin	Net/Signal	what it use for?
B1 (16)	VBAT_STACK	parallel add current/reduce voltage drop
B2 (17)	GND_STACK	ground
B3 (18)	3V3_STACK	add pin 3.3V again
B4 (19)	GND_STACK	ground
B5 (20)	5V_USB_STACK (optional/NC) send 5V from charge (if you don't use it, leave it blank)
B6 (21)	GND_STACK	ground
B7 (22)	5V_USB_STACK (optional/NC) send 5V from charge (if you don't use it, leave it blank)
B8 (23)	SPI_SCK (spare)	for sensor board // GPIO36 / 35 / 12
B9 (24)	SPI_MOSI (spare) for sensor board // GPIO35 / 37
B10 (25) SPI_MISO (spare) for sensor board // GPIO37 / 38
B11 (26) SPI_CS1 (spare)	for sensor board // GPIO10 / 11 / 14
B12 (27) SPI_CS2 (spare)	for sensor board // GPIO9 / 15
B13 (28) INT1 (spare) interrupt from sensor/RTC // GPIO4 / 5 / 6
B14 (29) INT2 (spare) interrupt // GPIO7 / 8
B15 (30) GPIO spare / SYS_RST (optional)	reset or other signal

GPIO1,2,19,20 for Sensor board ADC :
pin 37 : GPIO1 (ADC)
pin 38 : GPIO2 (ADC)
pin 39 : GPIO19 (ADC)
pin 40 : GPIO20 (ADC)
