use socker header pin 30 for all cubesat system ;

A (A1 → A15) :
Pin	Net/Signal what it use for?
A1	GND_STACK	ground
A2	VBAT_STACK	bat 7.4V send to Powerboard
A3	GND_STACK	ground (return current of VBAT)
A4	VBAT_STACK	parallel add current/reduce voltage drop
A5	GND_STACK	ground
A6	3V3_STACK	3.3V from Powerboard send to all board
A7	GND_STACK	ground
A8	3V3_STACK	parallel add current
A9	GND_STACK	ground
A10	I2C_SCL (GPIO2)	go to INA226/RTC/sensor
A12	I2C_SDA (GPIO1)	go to INA226/RTC/sensor
A11	GPS_RX (GPIO21)	ESP receive data (connect to GPS TX)
A13	GPS_TX (GPIO47)	ESP send (connect to GPS RX)
A14	PWR_EN / BUCK_EN (optional) command to open/close buck or open system (if you want)
A15	GND_STACK	ground

B (B1 → B15) :
Pin	Net/Signal	what it use for?
B1	VBAT_STACK	parallel add current/reduce voltage drop
B2	GND_STACK	ground
B3	5V_USB_STACK (optional/NC) send 5V from charge (if you don't use it, leave it blank)
B4	GND_STACK	ground
B5	5V_USB_STACK (optional/NC) send 5V from charge (if you don't use it, leave it blank)
B6	GND_STACK	ground
B7	3V3_STACK	add pin 3.3V again
B8	SPI_SCK (spare)	for sensor board // GPIO36 / 35 / 12
B9	SPI_MOSI (spare) for sensor board // GPIO35 / 37
B10	SPI_MISO (spare) for sensor board // GPIO37 / 38
B11	SPI_CS1 (spare)	for sensor board // GPIO10 / 11 / 14
B12	SPI_CS2 (spare)	for sensor board // GPIO9 / 15
B13	INT1 (spare) interrupt from sensor/RTC // GPIO4 / 5 / 6
B14	INT2 (spare) interrupt // GPIO7 / 8
B15	GPIO spare / SYS_RST (optional)	reset or other signal
